Dodaj słówko

Tłumaczenia alternatywne wpisuj po średniku, przecinku albo ukośniku, np. dom; mieszkanie; budynek. Quiz uzna każde z nich.

Tłumaczenia wpisujesz ręcznie. Przy słowach wieloznacznych dopisz kilka poprawnych wersji, np. dom; mieszkanie; budynek.

Import wielu słówek

Wklej listę, każda para w osobnej linii. Obsługiwane formaty: casa = dom; mieszkanie, casa - dom, casa → dom.

Synchronizacja Google Drive

Wersja przygotowana pod poziom drugi: ręczne przyciski zapisu i odczytu z ukrytego pliku aplikacji na Dysku Google. Do działania musisz wkleić własny Google OAuth Client ID i uruchamiać stronę z autoryzowanego adresu, np. localhost albo GitHub Pages.

Drive niepołączony. Najpierw wklej Client ID i kliknij „Połącz z Google Drive”.

Autosynchronizacja: wyłączona. Po włączeniu strona po połączeniu z Drive będzie wczytywać nowszy save i zapisywać zmiany po krótkiej chwili.

Plik backupu jest zapisywany jako habit_pixel_backup.json w ukrytym folderze aplikacji Google Drive, nie jako zwykły widoczny dokument na głównym Dysku.
